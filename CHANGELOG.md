# CHANGELOG

## 0.0.9 (2022-07-20)
* Adds `pkgmt test-md` command to test `README.md` files

## 0.0.8 (2022-07-19)
* Adds sample GitHub Actions files
* Adds sample test
* Adds `pytest`, `flake8` and `invoke` as dev dependencies
